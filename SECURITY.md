# Security

## Reporting a vulnerability

**English:** If you believe you have found a security issue, please **do not** open a public issue. Contact the repository maintainers privately (e.g. GitHub Security **Private vulnerability reporting** if enabled, or maintainer email if listed in the repo).

**中文：** 若认为发现安全问题，**请勿**公开发 Issue。请通过 GitHub「私人漏洞报告」（若已开启）或维护者预留的私下渠道联系。

## Notes

- This project embeds cryptography-related material in `src/generated/pem.ts` for operational parity with the upstream application. Treat repository access and forks accordingly.
- Compliance with JetBrains & other vendors’ terms of use is the responsibility of each operator.
